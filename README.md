# 📰 ArticleHub

ArticleHub is a modern, high-performance article discovery platform built with **React**, **TypeScript**, and **Vite**. The application features are dynamic filtering, and custom pagination.
Link to live app: https://article-hub-one.vercel.app/


## ✨ Features

* **Glassmorphism Header**: A floating, transparent navigation bar with backdrop-blur effects.
* **Dynamic Article Discovery**: Real-time search and category filtering.
* **Smart Pagination**: Efficiently browse large collections of articles.
* **Responsive Design**: Fully optimized for Mobile, Tablet, and Desktop views.
* **Type Safety**: Built with strict TypeScript for robust code quality.

## 🚀 Getting Started

### Prerequisites

* **Node.js**: Version 18.0.0 or higher
* **npm** or **yarn**

### 1. Installation

Clone the repository and install dependencies:

```bash
git clone [https://github.com/iamsukhe/article-hub](https://github.com/iamsukhe/article-hub)
cd article-hub
npm install
```

### 🛠 Run the project 

* **`npm run dev`**: Starts the Vite development server with Hot Module Replacement (HMR).
* **`npm run build`**: Runs the TypeScript compiler and builds the production-ready app to the `/dist` folder.
